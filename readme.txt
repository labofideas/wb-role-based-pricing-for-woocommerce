Dynamic WooCommerce pricing based on Pricing Groups with role-based eligibility.

== Description ==

WB Role Based Pricing for WooCommerce provides Phase 1 role-based pricing controls:

* Pricing Groups with priority and default rules
* Role -> Pricing Group mapping
* Product-level and variation-level pricing rules
* Category-level pricing rules
* Price calculation integration for product loops, product pages, cart, and checkout
* Order line item pricing audit metadata
* HPOS compatibility declaration

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate WooCommerce.
3. Activate **WB Role Based Pricing for WooCommerce**.
4. Configure under `WooCommerce -> Settings -> WB Role Based Pricing`.
5. Create groups under `WooCommerce -> Pricing Groups`.

== Frequently Asked Questions ==

= Does this support memberships/subscriptions? =

Not in Phase 1. Membership and subscription mappings are planned for Phase 2.

= Does this support variable products? =

Yes. Per-variation rules are supported.
